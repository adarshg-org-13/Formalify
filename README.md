## Formalify: A Formula sheet website

Formalify is a formula sheet website where you can practice your formulaes and learn them according to your subjets 
and classes as of now formialify has a support of classes 9 10 11 and 12 and Physics Chemistry and Maths as the 
subjetcs.

This website is made using Next.js framework

Install Next.js App

```bash
npx create-next-app@latest {your_folder_name}
```

This project uses some external modules 

```bash
npm install lucide-react
```

```bash
npm install katex react-katex
```

You can run the project locally by :-

```bash
npm run dev
```

To visit the website (local hosted):- http://localhost:3000
